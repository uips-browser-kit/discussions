# discussions

Central collaboration space for uips-browser-kit.

## Purpose

Use this repository for:

- scope discussions
- browser automation pain points
- proposal refinement
- idea exploration
- community questions

## Not intended for

- implementation PRs
- finalized specifications
- bug tracking for released artifacts

## How content moves

```
Discussion (here)
      ↓
Scope decision  →  open an issue in the relevant repo
      ↓
Implementation  →  maintainer PR (library / rules / snippets)
               or  contributor PR (testproject / testdata / docs / testharness-webapps)
```

When a discussion produces enough signal, open an issue in the relevant
repository. See [CONTRIBUTING.md](https://github.com/uips-browser-kit/.github/blob/main/CONTRIBUTING.md)
for which repositories accept direct contributions.
